Write a mobile application that draws basic graphical primitives on
the screen according to user input.
Description: Program should accept the following details from the user in
the first page : shape to be drawn(Drop down menu supporting circle, oval,
rectangle, triangle), color(drop down menu), filled(Radio button). On
accepting the inputs, the user should click on the Submit button that will
take the user to the next page. In the second page, the figure should be
drawn according to user choice made in the previous page. In the second
page, add a Back button, to go back the previous page.
